MUSIC SETUP INSTRUCTIONS
========================

To use the music menu system, organize your MP3 files in this folder structure:

F:\Website\
├── index.html
├── styles.css
├── script.js
├── logo.png
├── video.mp4
├── music.mp3 (default track - keep this)
└── music/
    ├── electronic/
    │   ├── ambient-flow.mp3
    │   ├── digital-dreams.mp3
    │   └── synthwave-night.mp3
    ├── classical/
    │   ├── piano-sonata.mp3
    │   ├── string-quartet.mp3
    │   └── orchestra-suite.mp3
    ├── jazz/
    │   ├── smooth-jazz.mp3
    │   ├── bebop-session.mp3
    │   └── late-night-blues.mp3
    └── rock/
        ├── guitar-riff.mp3
        ├── rock-anthem.mp3
        └── alternative-mix.mp3

HOW TO ADD YOUR OWN MUSIC:
1. Create the 'music' folder in your website directory
2. Create subfolders for each genre (electronic, classical, jazz, rock)
3. Place your MP3 files in the appropriate genre folders
4. Rename your MP3 files to match the names above, OR
5. Edit the musicLibrary object in script.js to match your file names

TO ADD NEW GENRES:
1. Create a new folder in the music directory
2. Add your MP3 files to that folder
3. Edit script.js and add your new genre to the musicLibrary object

Example of adding a new genre in script.js:
    'Pop': [
        { name: 'Pop Hit 1', file: 'music/pop/pop-hit-1.mp3' },
        { name: 'Pop Hit 2', file: 'music/pop/pop-hit-2.mp3' }
    ],
